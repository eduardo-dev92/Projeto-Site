RoadMap Dev

FrontEnd Developer

Os desenvolvedores frontend trabalham nas partes do site que o usuário vê e interage com ele. Isso também é às vezes chamado de desenvolvimento no lado do cliente ou no lado do navegador.

BackEnd DeveloperBackEnd Developer

Um desenvolvedor backend trabalha no servidor, banco de dados e APIs por trás de uma aplicativo web.A descrição do trabalho é diferente da de um desenvolvedor frontend, mas os dois papéis trabalham juntos para criar um trabalho que seja ao mesmo tempo funcional e amigável ao usuário.
